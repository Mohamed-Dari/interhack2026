## Clinic Demand Twin

MVP Streamlit per al repte Smart Demand Signals d'Inibsa.

```bash
cd "clinic-demand-twin"
pip install -r requirements.txt
streamlit run app.py
```

Si es vol executar la demo sintètica controlada:

```bash
CLINIC_TWIN_DATA_SOURCE=mock streamlit run app.py
```

El projecte complet és a `clinic-demand-twin/README.md`.
